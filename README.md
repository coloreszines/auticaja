**autiii v0.2** es un dispositivo inteligente autista y afectivo, diseñado como un sistema de apoyo emocional y afirmación neurodivergente. 
Surge desde la evolución de las *Cajas de Investigación* del archivo Redhumus, transformándose en una interfaz viva entre cuerpo, emoción y tecnología.

El dispositivo actúa como una **presencia cotidiana**: no es una máquina utilitaria ni una herramienta clínica, sino un compañero de pensamiento y respiro. Su comportamiento está inspirado en los **tamagotchis**, pero su propósito es otro: **sostener, afirmar y acompañar**.
